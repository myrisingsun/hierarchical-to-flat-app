# Hierarchical to Flat — ВОР → ЛЗК

Универсальный GUI-инструмент для трансформации иерархического списка **ВОР** (Ведомость объёмов работ) в плоский список **ЛЗК** (Лимитно-заборная карта).

---

## Назначение

Файлы ВОР хранят смету в виде трёхуровневой иерархии:

```
1   Раздел (Вентиляция дымоудаления)
1.1   Подраздел (Система ВД18)
1.1.1   Работа (Монтаж вентилятора...)
          Материал 1 (Вентилятор...)
          Материал 2 (Болты с гайками...)
          ...
```

Инструмент разворачивает эту структуру в плоскую таблицу — по одной строке на каждый материал с указанием, к какой работе, разделу и пункту сметы он относится.

---

## Результат (выходной файл)

| № работ | Раздел сметы | Наименование работ | Наименование материалов | Ед. изм. | Кол-во | Цена | Сумма |
|---|---|---|---|---|---|---|---|
| 1.1.1 | Вентиляция дымоудаления | Монтаж вентилятора... | Вентилятор... | шт | 1 | — | — |
| 1.1.8 | Вентиляция дымоудаления | Монтаж воздуховода... | Болты с гайками | кг | 17.08 | — | — |

## Требования

- Python 3.10+
- [openpyxl](https://openpyxl.readthedocs.io/) — `pip install openpyxl`
- tkinter (входит в стандартную поставку Python)

---

## Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/myrisingsun/hierarchical-to-flat-app.git
cd hierarchical-to-flat-app

# Установить зависимости
pip install openpyxl

# Запустить
python transform_vor.py
```

---

## Использование

1. Нажмите **Открыть…** и выберите Excel-файл ВОР (`.xlsx`)
2. Лист и колонка «Кол-во итого» определяются **автоматически**
3. При необходимости скорректируйте настройки колонок (A–Z)
4. Укажите путь для выходного файла (по умолчанию `*_ЛЗК.xlsx`)
5. Нажмите **Преобразовать**

### Авто-определение режима

| Структура листа | Режим | Результат |
|---|---|---|
| Работы + строки материалов | С материалами | Одна строка = один материал |
| Только работы (без материалов) | Только работы | Одна строка = одна работа, столбец «Наименование материалов» пуст |
